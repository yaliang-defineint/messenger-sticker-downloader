# Messenger Sticker Downloader

一個讓你一鍵下載 Facebook Messenger 聊天室中貼圖的 Chrome 擴充功能！  
支援下載會動且去背透明的原始貼圖檔案 (存檔為 `webp`)。

## 功能特點

* **完整還原**：下載下來的貼圖保持最原始的動態效果與透明背景。
* **極簡操作**：滑鼠對著貼圖點擊右鍵，選擇「下載此貼圖 」即可完成下載。
* **快速省流量**：直接提取瀏覽器已載入的貼圖資料，不佔用額外網路流量。

## 安裝說明

### 使用 Github Release

1. 前往本專案的 **Releases** 頁面，下載最新的 `extension.zip` 並解壓縮。
2. 打開 Chrome 瀏覽器，在網址列輸入 `chrome://extensions/` 進入擴充功能管理頁面。
3. 開啟頁面右上方角角處的 **「開發者模式」** 開關。
4. 點擊左上角的 **「載入未封裝擴充功能」** 按鈕，選擇你剛剛解壓縮出來的資料夾。
5. 重新整理你的 Messenger 網頁即可使用。

### 使用 chrome web store

可以到 [Chrome Web Store](https://chromewebstore.google.com/detail/messenger-sticker-downloa/jbfollfgeikfojphpgdnldmpeelomifb) 加入此功能來使用

* 本工具僅供個人學術交流與收藏喜愛貼圖使用，請勿將下載之貼圖用於商業用途。
* 本擴充功能不會收集、儲存或傳輸任何使用者個人資料，所有資料皆在本機處理。
