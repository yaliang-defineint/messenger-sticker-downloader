# Messenger Sticker Downloader

一個讓你一鍵下載 Facebook Messenger 聊天室中貼圖的 Chrome 擴充功能！  
支援下載會動且去背透明的原始貼圖，並提供 **WebP** 與 **GIF** 格式轉碼輸出。

## 功能特點

* **多格式支援**：可自由選擇儲存為原生動態 `WebP` 或廣泛相容的 `GIF` 格式。
* **完整還原**：下載下來的貼圖保持最原始的動態效果與透明去背背景。
* **極簡操作**：對貼圖點擊右鍵選擇「下載此貼圖」即可完成下載，並可透過擴充功能圖示隨時切換格式。
* **本機轉碼與隱私保護**：採用 Chrome Offscreen API 於背景進行影像轉碼，完全不佔用伺服器流量且不收集任何個人資料。

## 安裝說明

### 使用 GitHub Release

1. 前往本專案的 **Releases** 頁面，下載最新的 `extension.zip` 並解壓縮。
2. 打開 Chrome 瀏覽器，進入 `chrome://extensions/` 擴充功能管理頁面。
3. 開啟頁面右上方角角處的 **「開發者模式」**。
4. 點擊左上角 **「載入未封裝擴充功能」**，選擇解壓縮後的資料夾。
5. 重新整理你的 Messenger 網頁即可使用。

### 使用 Chrome Web Store

可直接造訪 [Chrome Web Store](https://chromewebstore.google.com/detail/messenger-sticker-downloa/jbfollfgeikfojphpgdnldmpeelomifb) 安裝使用。

## 使用須知

* 本工具僅供個人學術交流與收藏喜愛貼圖使用，請勿將下載之貼圖用於商業用途。
* 本擴充功能不會收集、儲存或傳輸任何使用者個人資料，所有圖片處理皆在本機完成。
